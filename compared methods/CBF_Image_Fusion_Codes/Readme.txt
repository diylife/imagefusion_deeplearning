This software release consists of an implementation of the algorithm described in the paper:

B. K. Shreyamsha Kumar, “Image fusion based on pixel significance using cross bilateral filter? Signal, Image and Video Processing, pp. 1-12, 2013. (doi:10.1007/s11760-013-0556-9)


Download Original manuscript from 

http://link.springer.com/article/10.1007/s11760-013-0556-9

or

http://dx.doi.org/10.1007/s11760-013-0556-9


Download draft manuscript from https://sites.google.com/site/shreyamsha in the Publications link.


It consists of the following files:

readme.txt: this file

images used: office256, gun, med256

main program: cbf_fusion_pgm

support .m files: cross_bilateral_filt2Df, gauss_ker2D, cbf_ieeeconf2011f,covarf, fusion_perform_fn, objective_fusion_perform_fn, imhist_fn, joint_hist_fn, per_extn_im_fn, sobel_fn


If you are using the Modified Fusion Artifacts (NABF) measure, modification of Petrovic's Fusion Artifacts measure, please cite:

B. K. Shreyamsha Kumar, multifocus and multispectral image fusion based on pixel significance using discrete cosine harmonic wavelet transform? Signal, Image and Video Processing, pp. 1-19, 2012, doi:10.1007/s11760-012-0361-x
